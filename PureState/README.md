El archivo `AdapFSM_Functions_V2.py` contiene las funciones necesarias para realizar las simulaciones del protocolo. El código del protocolo se encuentra dentro del archivo `AdapFSM_V2.py` . 
El código que realiza la simulación de protocolo dados los valores de la dimensión del estado d, el tamaño del ensamble N y la división de ensamble se encuentra dentro del archivo "Simulation.ipynb".
En la carpeta `SimulationData` se encuentran los archivos CSV que contienen los datos generados durante las simulaciones, estos datos corresponden a la infidelidad entre el estado original y el estado estimado en cada paso del protocolo.
La carpetas `23`,`24` y `25`, contienen los datos generados considerando las divisiones de ensamble $2/3 , 2/4$ y $2/5$ respectivamente.
El archivo `DataAnalysis.ipynb` contiene el código utilizado para la realización de los gráficos en cada caso, y los respectivos ajustes de curva.
